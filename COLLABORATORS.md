# Collaborator PR Policy

All contributions to this repository must follow these rules:

1. Every change must be submitted through a Pull Request (PR). Direct pushes to shared branches are not allowed.
2. PRs must be AI-generated or AI-assisted. PRs created without AI are not allowed.
3. Before opening a PR, the contributor must verify the project builds and compiles successfully (`npm run build`).

If a PR does not meet these rules, it should be updated before review.
